# Вариант № 09
### Задача:
Вам дается строка, представляющая собой число в двоичном виде. 
Ваша задача - удалить все нулевые биты в этой строке и вернуть соответствующее число

### Решение:
Создание директории для сборки
```bash
mkdir build && cd build 
```

Сборка проекта
```bash
cmake .. && make
```

Запуск тестов
```bash
./lab_tests
```

Запуск программы
```bash
./main
```
